# Working Documents Directory

**Purpose**: Temporary workspace for drafts, research notes, and work-in-progress documents.

**Policy**:
- Documents here are NOT authoritative
- Subject to cleanup (90-day retention for unused files)
- Promote useful documents to permanent locations
- Delete when no longer needed

**Structure**:
- `research-notes/`: Research and investigation notes
- `drafts/`: Document drafts before finalization

**Note**: Do not reference documents in this directory from governance or permanent documentation.

## Guidelines

### When to Use

- Drafting governance documents (ADRs, PDRs, POLs)
- Research notes and investigation findings
- Temporary analysis documents
- Scratch work and brainstorming

### When to Move Out

**Promote to permanent location when**:
- Document is complete and reviewed
- Content should be preserved long-term
- Referenced by other documentation

**Delete when**:
- Document is obsolete
- Content has been incorporated elsewhere
- Not referenced for 90+ days

### Cleanup Schedule

- **Quarterly review**: Identify unused documents
- **90-day retention**: Delete if not accessed
- **Promotion**: Move valuable content to permanent locations
